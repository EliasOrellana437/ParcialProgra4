1. ¿Qué valor agregado tiene el uso de Web Components a su proyecto?
Permite que el código sea más ordenado y reutilizable. Es como crear una "pieza de Lego" propia; podemos llevar la calculadora a cualquier otra página sin tener que volver a escribir todo el código, asegurando que el diseño no se desconfigure.

2. ¿De qué forma manipularon los datos sin recargar la página?
Usamos JavaScript para escuchar lo que el usuario escribe en tiempo real. Cuando cambias un número, el código hace la cuenta al instante y cambia el texto del resultado en la pantalla automáticamente, sin que la página tenga que reiniciarse o parpadear.

3. ¿De qué forma validaron las entradas de datos?
Lo hicimos de dos formas:

En el diseño: Configuramos los cuadros de texto para que solo acepten números y no letras.

En el código: Programamos una regla que verifica que no se pongan números negativos y que el número de personas sea siempre mayor a 0 (para que no dé error al dividir).

4. ¿Cómo manejaría la escalabilidad futura en su página?
La página está lista para crecer porque usamos un diseño modular. Si mañana queremos agregar una calculadora de impuestos o una de descuentos, simplemente creamos un nuevo componente y lo conectamos, manteniendo la misma estructura y colores de forma sencilla.
